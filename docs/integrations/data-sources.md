# Data Sources

## Oracle Integration

### Price Feeds
- **Chainlink**: Decentralized price oracles
- **Pyth**: High-frequency price data
- **Chronicle**: Ethereum-native price feeds

### Market Data
- **The Graph**: On-chain analytics and indexing
- **Dune Analytics**: Custom market analysis
- **DefiLlama**: TVL and protocol metrics

## Real-time Feeds

### CEX Integration
- **Binance**: Real-time order book and trade data
- **Coinbase**: Professional market data feeds
- **FTX**: Derivatives and futures data

### Social Sentiment
- **Twitter API**: Social sentiment analysis
- **Reddit**: Community sentiment tracking
- **Discord**: Real-time community insights

## Data Processing

All data sources are processed through Shogun's unified data pipeline with:

- **Normalization**: Standardized data formats
- **Validation**: Data quality and consistency checks
- **Aggregation**: Multi-source data fusion
- **Real-time Updates**: Sub-second data refresh rates
